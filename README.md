# TPUGuesser
Бот или приложение, которое выдает пользователю случайное фото какого-то места в корпусе ТПУ (библиотека, закуток, аудитория, памятник), снятое под необычным углом. Задача игрока — угадать и отметить на карте, где это находится. Чем ближе метка к реальному месту, тем больше очков. В конце недели — рейтинг "лучших следопытов".
